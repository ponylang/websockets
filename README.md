# websockets

A WebSocket server library for Pony, implementing the [WebSocket protocol](https://www.rfc-editor.org/rfc/rfc6455).

## Status

websockets is an in-development project. It's not usable yet.

## Installation

* Install [corral](https://github.com/ponylang/corral)
* `corral add github.com/ponylang/websockets.git --version 0.0.0`
* `corral fetch` to fetch your dependencies
* `use "websockets"` to include this package
* `corral run -- ponyc` to compile your application
